# Shoply - E-Commerce Landing Page

A clean, modern, and responsive e-commerce landing page sliced from a UI design.

## 🚀 Tech Stack
- **HTML5**
- **Tailwind CSS** (via CDN)
- **Vanilla JavaScript** (DOM Manipulation & Interactions)
- **FontAwesome** (Icons)

## ✨ Features
- Fully responsive design (Mobile, Tablet, Desktop)
- Interactive "Add to Cart" button with pop animation
- Hover effects on product cards and buttons
- Custom clipping mask on promotional banner

## 🛠️ How to Run
Since this project uses plain HTML, CSS, and JS with a CDN for Tailwind, you don't need any complex setup.
1. Clone this repository: `git clone https://github.com/yourusername/shoply-landing-page.git`
2. Open `index.html` in your favorite web browser.
3. (Optional) Use VS Code with the "Live Server" extension for a better development experience.
